### Run Backend (Spring Boot)
```cd backend/sneaker_store_backend

# Linux/macOS
./gradlew bootRun

# Windows
gradlew.bat bootRun
```

### Running the Frontend
```bash
# Navigate from the frontend folder
cd frontend 

# Install the project dependencies: 
npm  install 

# Start react development server 
npm start
```
